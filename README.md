# SOS Smartwatch App

**Autoren**: Hassan Khan, Mario Paia, Timon Polster

Dieses Projekt stellt eine Benutzeroberfläche für eine Smartwatch dar, basierend auf einem Figma Wireframe. Die UI simuliert eine Notfallanzeige und Vitaldatenübersicht.

### Startseite
Die Startseite bietet einen schnellen Überblick über:
- Vitaldaten teilen
- SOS-Schaltfläche
- Direktzugriff auf Herzfrequenz und Sauerstoff

### Notfallübersicht
In der Notfallliste werden wichtige Körperwerte mit Statusanzeige dargestellt.  
Jeder Eintrag zeigt:
- Namen des Vitalwerts
- Farblich markierten Status
- Aktuellen Messwert

### Detailseite für Vitaldaten
Beim Anklicken eines Vitalwerts öffnet sich eine Detailseite mit:
- Groß angezeigtem aktuellen Wert
- Einem einfachen Liniendiagramm
- Zusatzinfos wie Normalbereiche, Trend und Zeitpunkt der letzten Messung

### Smartwatch-Rahmen
Die UI wird in einem simulierten Smartwatch-Gehäuse gezeigt, inklusive:
- Gehäuse mit abgerundeten Kanten
- Bildschirm mit dunklem Verlauf
- Seitliches Uhrenband für die Darstellung
